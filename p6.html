PROJECT 04 Reinforcement Learning

HVAC 자동 제어 시스템 DQN-based HVAC Control

프로젝트 개요

심층 강화학습(DQN)을 활용해 HVAC 설정온도를 자동 조정하는 제어 시스템입니다. 겨울철 난방 운전 시 실내 쾌적성 유지와 에너지 비용 절감이라는 상충되는 목표 간의 최적 균형 정책을 학습하여 적용합니다.

주요 역할 및 성과

  • RL 환경 설계: 상태(실내/외기온도), 행동(난방 설정온도 ±2°C), 보상(쾌적도+비용페널티) 정의
  • DQN 모델 구현: PyTorch 기반 Q Network/Target Network 이중 구조 및 Replay Buffer 적용
  • 온라인 학습 파이프라인: Airflow 기반 15분 주기 데이터 수집-학습-추론 자동화
  • 에너지 최적화: 외기온도 급감 시 선제적 난방 강화, 일사량 충분 시 설정온도 하향으로 과난방 방지
